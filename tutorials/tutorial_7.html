Details - Custom Harmony

If you want to design your own harmonic rythm, this is the tutorial for you!

  1. Generate a new song!
  2. Create a custom song structure with just a single verse1. Set "Custom Harmony Indices" to 1 to modify the harmonic rythm to a custom one. Zero means "use default".
  3. Add a custom harmony that changes scale.
  4. Add voice line constraint for melody to end on something. This is an optional step but sometimes a phrase sounds a lot better if you end on the tonic or not on the tonic.

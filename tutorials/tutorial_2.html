Render a Midi File

This tutorial will show you how to generate a Midi file.

  1. Open the "Song Settings" window
  2. Make sure that you select the "Song" tab
  3. Change the "Seed" property to anything (your name?) or generate a random by pressing the small "R" button
  4. Open the "Export" window and select the "Midi" tab
  5. Click the "Export Midi" button and wait for the Midi file to generate
  6. Click on the "Result Link" to download the file!
  7. To generate a Wav, Mp3 or Ogg file from the Midi, you can use a program called Synthfont

Domains - Phrase Group Types

It is possible to control the generated song even more by modifying values in the "Domains" tab in the "Song Settings" dialog.

You can for example change the instruments to use for the different voices (melody, bass and inner), tempo range, the exact song structure to use and a lot more!

  1. Generate a new song!
  2. Select the "Domains" tab
  3. Set the "Tempo Range" property to something low such as 40 50. This will give a random tempo between 40 and 50 beats per minute.
  4. Scroll down to the "Phrase group types" list. Here you can change the possible phrase types and corresponding likelihoods to use in your song. You can remove or add as many phrase types you want (and create your own if you go to the "Details" tab). For this tutorial, remove everything but "Modulate + complete", "Tonicize + complete", "Modulate + modulate back".
  5. Export and play the song! Now, all the phrases (except the fillers between phrase groups, intro and end) move between different scales.

You can modify what similar scale to move to by changing "Major modulation/tonicization targets" and "Minor modulation/tonicization targets". Test for example to move to supertonic in major and subtonic in minor. Remember that you can change the scale type in the "Parameters" tab.

When you have deceptive cadences, the root to use as deceptive chord is modified with "Major deceptive root likelihoods" and the minor variant.
